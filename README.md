Photoscrub
==========

Scrub, overwrite and debug exifdata.

```
USAGE:
    photoscrub -i <input-file> <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i <input-file>

SUBCOMMANDS:
    help         Prints this message or the help of the given subcommand(s)
    list
    overwrite
    scrub
```
